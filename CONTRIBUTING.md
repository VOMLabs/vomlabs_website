## How to Contribute

1. **Fork the Repository** — Click 'Fork' to create your own copy

2. **Create a Branch** — Use a descriptive name:

   ```bash
   git checkout -b feature/my-new-feature
   ```

3. **Make Your Changes** — Write clear, concise code and include tests

4. **Commit Your Changes** — Follow [Conventional Commits](https://www.conventionalcommits.org/):

   ```bash
   git commit -m "feat: add feature X"
   ```

5. **Push to GitHub**

   ```bash
   git push origin feature/my-new-feature
   ```

6. **Create a Pull Request** — Submit against `main` with a description
